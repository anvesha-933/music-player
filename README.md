# 🎵 Music Player

A modern and responsive Music Player built using **HTML, CSS, and JavaScript**. It provides an intuitive interface for playing music with essential controls and a clean glassmorphism-inspired design.

## 📌 Features

- ▶️ Play & Pause music
- ⏭️ Next Song
- ⏮️ Previous Song
- 🎼 Dynamic Song Information
- 🖼️ Album Cover Updates
- 📊 Interactive Progress Bar
- ⏱️ Current Time & Total Duration
- 🔊 Volume Control
- 📃 Playlist
- 🎨 Modern Glassmorphism UI
- 📱 Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome Icons
- Google Fonts (Poppins)

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/music-player.git
```

2. Open the project folder.

3. Open `index.html` in your browser or run it using **Live Server** in VS Code.

## 🎮 Controls

| Action | Description |
|--------|-------------|
| ▶️ Play | Starts the selected song |
| ⏸️ Pause | Pauses the current song |
| ⏮️ Previous | Plays the previous song |
| ⏭️ Next | Plays the next song |
| 📊 Progress Bar | Seek to any part of the song |
| 🔊 Volume Slider | Adjust the playback volume |
